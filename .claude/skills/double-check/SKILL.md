Double-check the last work, whether it be a plan, design, or implementation, ensuring all aspects have been fully considered.

If last work was an implementation from a plan or design, check each aspect has been correctly and completely implememted as specified, correcting when not the case.

If last work was a plan or design, check for completeness, correctness, and potential issues.

If double-check was just completed, triple-check, and so on, until you are confident in the work.
